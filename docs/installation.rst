============
Installation
============


Stable release
--------------

To install High VRE capacity expansion, run this command in your terminal:

.. code-block:: console

    $ pip install high_vre_capacity_expansion

This is the preferred method to install High VRE capacity expansion, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for High VRE capacity expansion can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/sriharisundar/high_vre_capacity_expansion

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/sriharisundar/high_vre_capacity_expansion/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/sriharisundar/high_vre_capacity_expansion
.. _tarball: https://github.com/sriharisundar/high_vre_capacity_expansion/tarball/master
